# 🚖 OLA Data Analyst Project | Power BI Dashboard

## 📌 Project Overview
This project is an end-to-end Data Analytics project developed using Power BI. The dataset contains 1,00,000 simulated OLA ride bookings from Bengaluru city for one month. The project focuses on analyzing ride patterns, revenue, cancellations, customer behavior, and vehicle performance through interactive dashboards.

---

## 📊 Dataset Information
The dataset includes the following details:

- Date and Time
- Booking ID
- Booking Status
- Customer ID
- Vehicle Type
- Pickup and Drop Locations
- Average Vehicle Arrival Time (VTAT)
- Average Customer Arrival Time (CTAT)
- Cancellation Details
- Incomplete Ride Details
- Booking Value
- Ride Distance
- Driver Ratings
- Customer Ratings

## ✨ Features / Highlights
- 📊 Built an interactive Power BI dashboard using 1 lakh ride booking records.
- 📈 Analyzed ride volume trends across weekdays, weekends, and match days.
- 🚖 Evaluated booking performance across multiple vehicle categories.
- 💰 Monitored revenue trends and identified top customers by booking value.
- ❌ Analyzed customer and driver cancellation patterns and their reasons.
- ⭐ Measured driver and customer satisfaction through ratings analysis.
- 📍 Analyzed pickup and drop locations across 50 Bengaluru areas.
- ⏱️ Tracked operational metrics such as Vehicle Arrival Time (VTAT) and Customer Arrival Time (CTAT).
- 📌 Designed interactive dashboards with filters, slicers, cards, charts, and drill-down capabilities.
- 📖 Converted raw ride booking data into meaningful business insights and visual stories.

## 🎯 Project Objective
The objective of this project is to demonstrate how data analytics can be used to understand ride booking trends, identify cancellation patterns, analyze revenue, evaluate customer and driver performance, and generate actionable business insights for a ride-hailing company like OLA.
